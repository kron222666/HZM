# HZM: Hierarchical Zero Algebra

**Иерархическая алгебра нулей** — расширение множества вещественных чисел, в котором деление на ноль становится корректной, замкнутой операцией с сохранением иерархии уровней сингулярности.

## 📖 О проекте

В классической арифметике деление на ноль не определено, что приводит к неопределённостям, разрывам в вычислениях и ограничивает моделирование сингулярностей.  
**HZM** вводит **иерархические нули** $0_k$ и **иерархические бесконечности** $\pm\infty_k$ с дискретным уровнем $k\in\mathbb{N}$, а также поглощающий элемент $\bot$.

### Основные свойства

- **Замкнутость** – любая арифметическая операция над элементами HZM даёт элемент HZM.
- **Иерархия** – уровень $k$ служит количественной мерой глубины сингулярности.
- **Необратимость** – операции с нулями и бесконечностями, как правило, необратимы (потеря информации), что моделирует термодинамическую стрелу времени.
- **Совместимость** – при $k=0$ поведение совпадает с обычной арифметикой вещественных чисел.
